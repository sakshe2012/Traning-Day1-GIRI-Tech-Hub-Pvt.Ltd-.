# Java Fundamentals Practice Programs

## 📌 Overview
This repository contains a collection of beginner-friendly Java programs designed to strengthen core programming fundamentals and develop logical thinking skills. The exercises focus on Java basics such as data types, operators, arithmetic calculations, conditional statements, number manipulation, and problem-solving through practical coding examples.

These programs were implemented as part of hands-on technical training to gain practical experience with Java syntax and programming concepts.

---

## 📚 Topics Covered

- Java program structure
- Variables and basic data types
- User input using Scanner class
- Arithmetic operations
- Conditional statements (`if`, `if-else`)
- Number-based logic
- Mathematical calculations
- ASCII operations
- Salary and percentage calculations
- Problem-solving and logical implementation

---

## 📝 Programs Included

### Basic Java Programs
1. Display the message: `"This is first java program"`
2. Input and display all basic data types
3. Perform arithmetic operations on two numbers
4. Find the third angle of a triangle
5. Calculate total marks and percentage of five subjects
6. Calculate Simple Interest
7. Swap two numbers using a third variable
8. Swap two numbers without using a third variable
9. Print ASCII value of a character
10. Convert seconds into hours, minutes, and seconds

### Number Logic Programs
11. Check whether a number is a Neon Number
12. Toggle character case using ASCII values
13. Check whether a number is even or odd
14. Check whether a number is positive, negative, or zero
15. Check divisibility by 5 and 11

### Triangle Programs
16. Check whether a triangle is valid
17. Determine triangle type:
   - Equilateral
   - Isosceles
   - Scalene

### Employee & Salary Programs
18. Calculate Gross Salary using:
   - HRA (House Rent Allowance)
   - DA (Dearness Allowance)
   - Salary conditions
